## Report a security issue

Report security issues privately through [GitHub's security advisory form](https://github.com/tago-io/sdk-js/security/advisories/new). Do not report security vulnerabilities on the public issue tracker.

## Vulnerability coordination

Confirmed vulnerabilities are coordinated through the [GitHub Security Advisory](https://docs.github.com/en/code-security/security-advisories) opened for the issue. Affected parties include the reporter, users of `@tago-io/sdk`, and downstream maintainers where relevant.

## Security advisories

Security fixes are disclosed through [GitHub release notes](https://github.com/tago-io/sdk-js/releases) and the [npm advisory database](https://www.npmjs.com/advisories) when applicable.
